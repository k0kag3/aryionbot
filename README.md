## Usage

### Start

```shell
yarn dc:start
```

### Stop

```shell
yarn dc:stop
```

### Update

```shell
yarn dc:update
```

### Show logs

```shell
docker-compose logs -f
```

## Dev

### Launch dev server

```shell
yarn dc:dev
```

### Cleanup

```shell
docker-compose down --rmi local
```
